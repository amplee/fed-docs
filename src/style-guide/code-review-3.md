具体到每个项目，每个成员，在当前季度的随机5到10次提交。

如果该提交违背考核点，则打 ❌ ，否则打 ✅ 。<br />
满分💯，打 ✅不扣分，打 ❌扣除相应分数。
最终综合所有项目，所有提交点得出综合评分。

打分板应根据 [评估模板](/style-guide/code-review-1.html) 的结果进行打分。
